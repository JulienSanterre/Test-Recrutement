Vous pouvez trouver via le fichier Insomnia.json les données exporté des différentes requêtes utilisable.

Le logiciel insomnia : https://insomnia.rest/

il vous suffira d'importer ce fichier sur insomnia via le menu :
Application > Préférences -> Onglet Data -> Import Data > From File 
Et de choisir le fichier Insomnia.json
